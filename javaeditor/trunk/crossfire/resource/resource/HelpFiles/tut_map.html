Basics for designing a map

Once you have opened a map, you can start to shape and modify it. You can do the following actions with your mouse:

  • left mouse button: select game object(s)
  • right mouse button: insert an game object
  • middle mouse button: delete an game object

Selecting game object

To select a game object, left click on it. You can do so either on the Map View or in the Selected Square Display to the right. If you want to select a group of squares on the map (e.g. for copy/paste actions), left click and drag. You will notice a rectangular area getting highlighted as you drag.

Inserting game objects

To insert a new game object, you must first select one in the Object Chooser to the left. Right click to insert the selected game object into the map. You can also right-click into the Selected Square Display in order to insert the game object at a specific place in a pile of game objects. You can drag the mouse on the Map View in order to insert the same game object on many map squares.

To insert into the inventory of a game object (often needed for containers and monsters), proceed as follows:
In the Object Chooser, select the object you want to insert. On the Map View or Selected Square Display, select (highlight) the target game object that you want to insert into. Press the "Add Inv" button to insert.

There is a convenience function for inserting walls into a map: The "auto joining"-feature. Activate the checkbox in the menu "Map->Autojoin" to enable it. Whenever you insert or delete a wall now, any walls of the same type will automatically join to form a continuous solid wall. This saves you the hassle of switching between all the different wall archetypes.

Deleting game objects

To delete a game object, simply middle click on it. You can drag the mouse to delete multiple game objects. Deleting specific game objects out of a stack is possible by middle-clicking in the Selected Square Display.
