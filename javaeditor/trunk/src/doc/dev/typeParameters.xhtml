Gridarta Type Parameters

This is a list of all classes that are sometimes used as base types for generics type parameters, along with the suggested type parameter name.

  • G: GameObject
  • R: Archetype
  • A: MapArchObject
  • M: MapModel
  • T: Type - Some abstract specific data type, usually in non-collection sense.
  • E: Element - Some unspecific data type, usually in collection sense.
